# Video Center
Install a personal video web with Django

## Motivation
製作一個個人化的影片播放網站，這一個網站是私人的，不被公開的。使用者必須使用帳號密碼才可以登入。再這個網站裡可以撥放 HTML5 video tag 格式，讓其使用者可以分享自己的影片。

## Network framework
* Django 1.11
* Python3
* uWSGI
* Nginx
* Ubuntu 16.04
